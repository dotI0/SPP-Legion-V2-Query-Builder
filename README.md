# SPP-Legion-V2-Query-Builder (Alpha)

Tutorials and further information are available in the Wiki tab.  
[Wiki](https://github.com/dotI0/SPP-Legion-V2-Query-Builder/wiki)

Graphical scripting tool for SmartAI and Conditions.  
The tool is customizable and is provided with SPP Legion V2 specific documentation and declarations by default.  
The implementation of additional TrinityCore functionality is planned for future releases.  
The tool is currently in an alpha state and may contain bugs.  
Please use the Issues tab to submit bug reports and feature requests. 

**To use this software effectively, a solid understanding of how SmartAI scripts function in TrinityCore is required.**  
**Users should be familiar with the structure of SmartAI, including how scripts are created, configured, and modified within the database.**

**This tool is designed to simplify and streamline the scripting process, but it does not replace the underlying knowledge or decision making involved in creating SmartAI scripts.**

**If you are not yet familiar with SmartAI or TrinityCore scripting concepts, it is strongly recommended to review the resources listed below before using this tool.**

https://trinitycore.atlassian.net/wiki/spaces/tc/pages/2130108/smart_scripts
